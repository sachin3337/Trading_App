

A fully responsive clone of the Zerodha online trading platform built for learning and practice purposes.
This project replicates the UI/UX and some basic functionality of the original platform for educational use only.

🚀 Features
🔹 Clean and modern landing page
🔹 Responsive design (mobile + desktop)
🔹 Recreated UI components inspired by Zerodha
🔹 Login page & signup page
🔹 Dashboard mockup for user portfolio
🔹 Smooth animations & transitions
🔹 Reusable components
🔹 Organized folder structure
🛠️ Tech Stack

Frontend

HTML5
CSS3 / TailwindCSS / Bootstrap (choose what you used)
JavaScript / React.js

Backend (optional)

Node.js + Express
Authentication mock / API mock
Database (MongoDB / Firebase)
